# Spring Boot Vue OAuth2 Demo

# <h4>Setting Backend
* Create postgresql database
* Setting username and password
* Run backend
	```bash
    cd backend
    ```
    ```bash
    .\mvnw spring-boot:run
    ```
 
 # <h4> Setting Fronted
 * Run frontend
 	```bash
    cd frontend
    ```
    ```bash
    npm install && npm run serve
    ```
